This Repo is a Phaser.js game made in JavaScript. The goal of this game is to be deployed and allow users across the web to play against an enemy AI that gets smarter each time someone plays the game. 

This refactoring and deploying is a personal project by Jonah Pattison.

The original game was made with Pygame in Python as a group project for Portland State's AI course. 

That original project is here: https://github.com/JPatts/AI-Research-Project
